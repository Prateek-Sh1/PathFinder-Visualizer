# 🔍 Pathfinding Visualizer

An interactive web-based visualization tool for understanding pathfinding algorithms like **Breadth-First Search (BFS)** and **Depth-First Search (DFS)**.

---

## 🚀 Features

* Visualize BFS (Shortest Path in unweighted grid)
* Visualize DFS (Traversal and exploration)
* Interactive grid for creating obstacles (walls)
* Real-time animation of visited nodes and final path
* Color-coded representation for better understanding

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🎯 What I Learned

* Graph traversal algorithms (BFS, DFS)
* Grid-based problem solving
* DOM manipulation and animations
* Visualizing algorithms for better understanding
